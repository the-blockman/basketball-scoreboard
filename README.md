# 🏀 Basketball Scoreboard Web App

A simple, interactive basketball scoreboard built with **HTML, CSS, and JavaScript**.  
Designed to simulate a real-game experience with features like a game clock, shot clock, score tracking, and user controls.

🔗 **Live Demo:**  
https://basketball-scoreboard-javasc.netlify.app/

---

## 🚀 Features

- ⏱️ **Game Timer**
  - Select game duration (5, 7, 10, or 12 minutes)
  - Timer starts only when a new game begins
  - Pause and resume functionality

- 🏀 **Shot Clock**
  - 24-second shot clock
  - Automatically resets on score

- 🔢 **Score Tracking**
  - Add points (+1, +2, +3) for both teams
  - Reduce score (-1) for correction
  - Prevents negative scores

- 🟢 **Leading Team Highlight**
  - Leading team score is highlighted

- 🔄 **Game Controls**
  - New Game resets everything
  - Timer selection locked during gameplay

---

## 🛠️ Tech Stack

- **HTML5** – Structure
- **CSS3** – Styling and layout
- **Vanilla JavaScript** – Logic and interactivity

---

## 🎮 How to Use

1. Select your desired game time
2. Click **"New Game"**
3. Use:
   - `+1`, `+2`, `+3` to add points
   - `-1` to correct mistakes
4. Use **Pause/Resume** to control the timer
5. Shot clock resets automatically on scoring
